# 確定要搬家了

確定要搬家了  
就在這個禮拜日 從彰化市崙美路搬到和美鎮和頭路去  
而搬家的理由是 為了節省開銷  
因為目前在崙美路的這個家  
是由鐵皮屋搭成的100坪工廠  
月租費2萬多  
在加上這裡的水費電費都是採記很高的費率  
又快要到達使用冷氣的高峰期  
電費又要暴漲  
所以決定要搬到比較鄉下的和美鎮透天  
只是  
搬去之後  
就會變得很無聊了  
上下學也不能其腳踏車了  
相對之下會變得很不自由  
到時候生活一定會更無聊  
到還希望可以趕快上大學......
