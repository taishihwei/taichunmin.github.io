# 第一天開學就睡過頭了

最近這三天  
我發現一個奇怪的定律  
新生訓練前一天  
睡覺時間是三點，早上六點半起來  
新生訓練第二天前  
睡覺時間是兩點，早上七點半起來  
開學前天  
睡覺時間是一點，早上...............八點半＝ ＝  
第一天的勞作服務就遲到了  
還好學長沒有記遲到  
還是記成正常來掃地  
慶幸小組長的仁慈xd  
話說  
早上好像有兩個人打電話叫我起床  
可是  
我的手機好像收訊不良  
一通未接來電都沒有＝ ＝

只要我的手機處於無訊號的情況

所以  
那兩個同學就叫到了我弟  
我弟  
現在還在家中悠閒  
被吵起來他應該會很不爽吧  
真糟糕阿：p
