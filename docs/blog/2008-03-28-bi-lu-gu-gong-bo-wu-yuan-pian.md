# 畢旅-故宮博物院篇

一開始走到故宮的階梯之後  
去拍了幾張照  
就又叫我們往下走  
這時候我就覺得很奇怪  
心裡想: 不是要去故宮嗎 怎麼要往下走??  
雖然懷疑但還是跟著走到至善園  
看到至善園的風景真的是很好  
然後就做我的白日夢...  
在等了很久很久之後  
終於要進故宮了  
當然也在這段期間得知目前故宮太多人等等在進的事  
進故宮後又在一樓等了很久  
因為領隊說要領 故宮提供的專門解說 搭配的收音機  
但是到後來啥都沒有  
只好無奈的入場了

  
進去之後領隊就馬上帶我們到2樓  
去看瓷器  
但是我看半天  
還是對其不怎麼有印象  
因為我根本不知道在看啥...  
我在記念品區買了三個書籤  
金色的生肖鍍金純銅書籤  
真的很好看  
後來  
集合時間到了以後  
我們這一群22車的人又在外面呆站了很久  
到後來等不下去就由領隊先帶回去  
留下阿法組長下來等  
然後遊覽車再轉回去載  
所以 慢了大家很多
