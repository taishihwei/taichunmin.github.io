# printf函式內使用 * 字號

```cpp
/*
這是某一天我弟給我看的程式碼
可以拿來輸出米字號三角形
奧秘在於 %s 中間的 * 字號
表示不確定顯示幾位時使用
要記得傳一個整數型態的變數來代表位數 
//*/

#include <stdio.h>

int main()
{
    int i=0;
    for(i=1;i<=10;i++)
        printf("%.*s\n", i, "****************");
    return 0;
}
```
