# 蝸牛寒期訓練營-二籌

新訓二籌  
是粗流已經確定  
然後分配各組工作報告的時間  
可是這個時候的我們  
粗流根本還沒完全確定出來  
所以二籌變成在確認粗流  
分派給各組的工作真的是一團亂  
會議的主持人昱丞也因為沒有總籌的經驗  
沒有辦法指派工作下去  
所以就感覺整個二籌就只是決定出粗流…  
一開始我們二籌也開得太嚴肅了  
總覺得大三他們開會都可以開的很愉快  
可是我們自己開會都可以弄的非常無聊  
那一天開會弄的有點晚  
真的是對我爸媽很抱歉…  
讓他們多等了一個多小時
